# 豆花语音输入：基于豆包APP的语音输入助手
> 🎙️ MacOS 豆包语音输入增强工具 - 一键语音转文字，按住说话自动插入，让豆包桌面版语音输入更高效

这是一个为 macOS 平台量身定制的辅助工具，旨在优化豆包客户端的语音输入体验。通过全局快捷键托管和智能的剪贴板保护机制，它能将豆包原本“唤醒 -> 说话 -> 手动回车 -> 粘贴”的繁琐流程，转化为极其丝滑的**“按住说话、松开自动上屏”**体验。

## ✨ 核心特性

- 🎙️ **按住即说**：长按右侧 Command 键唤醒豆包录音，松开自动发送文字到当前输入框。
- 🛡️ **剪贴板保护**：语音输入过程不会覆盖你原本复制的内容。程序会在录音前备份剪贴板，并在文字上屏后自动恢复。
- 🚫 **防误触机制**：内置长按阈值（默认 250ms），正常短按 `Cmd+C` / `Cmd+V` 等快捷键绝不会触发语音。
- 🤫 **静默取消**：如果你长按了按键但没有说话，松开后悬浮窗会自动消失，不会产生任何多余的字符或粘贴动作。
- 🎛️ **可视化控制台**：自带轻量级深色 Web 控制台，支持实时调整延迟参数和查看运行日志。
- 🏥 **高可用健康守护**：后台自动监控 macOS 键盘监听器状态，遭遇系统睡眠或异常中断时自动恢复；同时提供一键“重启监听器”功能，彻底告别“用久了突然失效”的烦恼。

## 🚀 快速开始

### 1. 前置准备
- 确保你已经安装了 macOS 版的 **豆包客户端**。
- 在豆包客户端的设置中，进入 **设置 -> 快捷键 -> 语音输入**，将语音输入的快捷键修改为 `Option + D`（这是为了避开类似 iTerm2 `Ctrl + D` 退出终端的冲突，也是本程序默认模拟触发的快捷键）。
- 确保你的 Python 环境为 Python 3.8 或以上。

> **注意：** 必须确保豆包里的快捷键和本程序的配置一致。如果你改成了别的键，需要同步修改 `~/.douhua_voice/config.json` 里的 `trigger_combo` 字段。

### 2. 安装依赖
克隆本项目到本地，并安装必要的依赖：
```bash
pip install -r requirements.txt
```

### 3. 启动服务
在终端中运行以下命令：
```bash
python3 main.py
```
> **⚠️ 首次运行注意**：
> macOS 会请求“辅助功能(Accessibility)”权限，因为程序需要监听键盘按键和模拟输入。请在 `系统设置 -> 隐私与安全性 -> 辅助功能` 中，允许你的终端（如 Terminal、iTerm、Cursor 等）控制电脑。

启动成功后，程序会自动在浏览器中打开 Web 控制台：`http://127.0.0.1:8899/`。

## 🕹️ 如何使用

1. **说话**：在任何需要打字的软件里（微信、浏览器、IDE等），**按住**键盘上的 `右 Command` 键不放，此时豆包的语音悬浮窗会弹出并开始录音。
2. **上屏**：说完话后，**松开**按键。程序会自动触发豆包识别，并将结果直接插入到你当前的输入框中。
3. **取消**：如果你按住了按键但不想说了，直接松开即可，什么都不会发生。
4. **一键修复失效**：如果在长期挂机后，发现长按按键没有任何反应，请打开 `http://127.0.0.1:8899/` 点击 **“重启监听器”** 按钮即可满血复活（通常是因为终端开启了“安全键盘输入”或者系统挂起了事件导致）。

## ⚙️ 进阶配置 (Web 控制台)

由于不同电脑的性能和不同 App 的响应速度存在差异，如果你在使用中遇到卡顿或粘贴异常，可以通过访问 `http://127.0.0.1:8899/` 来微调参数：

- **`hold_threshold_ms`** (默认 250): 长按阈值。按住时间超过此值才触发语音，避免和正常快捷键冲突。
- **`normal_delay_ms`** (默认 260): 松开按键后，等待豆包完成最后一句话识别的缓冲时间。
- **`submit_settle_timeout_ms`** (默认 1800): 等待豆包将识别结果放入剪贴板的超时时间。如果在网络较差时经常识别失败，可以适当调大此值。
- **`post_submit_grace_ms`** (默认 380): 结果上屏后的缓冲时间，确保文字真正落入输入框后再恢复剪贴板。

*修改参数后点击“保存配置”即刻生效，无需重启服务。*

## 🛠️ 技术栈

- `pynput`：全局键盘监听与按键模拟。
- `subprocess` (`pbcopy`/`pbpaste`)：无感操作 macOS 剪贴板。
- `http.server`：内置超轻量级 Web 服务。
